PostgreSQL Storage Visualizer

Select a table or index from the sidebar to explore its internal structure

Part of boringSQL toolkit

Features:

  • Table inspection - heap pages, tuple layout, MVCC visibility
  • Index visualization - B-tree structure, page density, bloat
  • HOT update demo - see heap-only tuples and ctid chains
  • Row finder - locate rows by PK or TID

Uses built-in pageinspect and pgstattuple extensions.
